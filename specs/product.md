# Product Specification: RetroFeed

## Vision
A high-signal, retro-style terminal intelligence aggregator that pulls from diverse sources (RSS, HN, Reddit, and eventually Telegram and Finance) to provide a unified, prioritized feed.

## Core Features
- **Aggregated Stream**: Real-time unified view of multiple data sources.
- **Scoring Engine**: Priority-based ranking using source weights, keywords, and recency.
- **Alerts Engine**: Threshold and pattern-based triggering for high-priority items.
- **Matrix-Style UI**: Web-based terminal presentation.
- **Deterministic Pipeline**: Every stage from ingest to alert is testable and predictable.
